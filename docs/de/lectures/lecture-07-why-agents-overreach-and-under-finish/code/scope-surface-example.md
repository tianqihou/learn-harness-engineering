# Scope-Oberfläche Beispiel

Aufgabe:

- Indexierung zur Electron Knowledge-App hinzufügen

Schlechte Scope-Form:

- "Indexierung implementieren"

Bessere Scope-Form:

- Importierte Dokumente parsen
- Dokumente in Chunks aufteilen
- Chunk-Metadaten persistieren
- Indexierungsstatus in der UI anzeigen
- Eine Neu-Indexierungs-Aktion hinzufügen
