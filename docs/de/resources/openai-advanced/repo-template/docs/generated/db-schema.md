# Datenbankschema

Verwenden Sie diesen Ordner für generierte oder abgeleitete Artefakte, die Agenten
inspizieren können sollten, ohne sie aus dem Code rekonstruieren zu müssen.

## Quelle

- Generiert aus: `[Befehl oder Quellpfad]`
- Zuletzt aktualisiert: `YYYY-MM-DD`

## Hinweise

- Generierte Abschnitte nicht manuell bearbeiten.
- Diese Datei neu generieren, wenn sich das zugrunde liegende Schema ändert.
