# Grundüberzeugungen

- Das Repository ist das System of Record für Agenten.
- `AGENTS.md` ist ein Router, keine Enzyklopädie.
- Verifikationsnachweise sind wichtiger als Selbstvertrauen.
- Eine begrenzte Aufgabe ist besser als viele halbfertige Aufgaben.
- Wiederholtes menschliches Feedback sollte zu wiederverwendbaren Harness-Regeln werden.
- Aufräumen und Vereinfachung sind Teil des Auslieferns, keine Nachgedanken.
- Wenn ein Agent eine Tatsache nicht im Repo finden kann, behandeln Sie diese
  Tatsache als operationell nicht verfügbar.
