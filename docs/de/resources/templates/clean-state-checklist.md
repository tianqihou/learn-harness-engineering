# Clean-State-Checkliste

- [ ] Der Standard-Startpfad funktioniert noch.
- [ ] Der Standard-Verifikationspfad läuft noch.
- [ ] Der aktuelle Fortschritt ist im Fortschrittslog dokumentiert.
- [ ] Der Feature-Status entspricht dem, was tatsächlich bestanden vs. unverifiziert ist.
- [ ] Kein halbfertiger Schritt bleibt undokumentiert.
- [ ] Die nächste Session kann ohne manuelle Reparatur weiterarbeiten.
