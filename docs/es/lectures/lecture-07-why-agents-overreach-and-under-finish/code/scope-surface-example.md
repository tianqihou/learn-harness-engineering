# Ejemplo de Superficie de Alcance

Tarea:

- Añadir indexación a la app de conocimiento Electron

Forma de alcance mala:

- "Implementar indexación"

Forma de alcance mejor:

- Analizar documentos importados
- Dividir documentos en fragmentos
- Persistir metadatos de fragmentos
- Mostrar estado de indexación en la UI
- Añadir una acción de reindexación
