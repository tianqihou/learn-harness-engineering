# Ejemplo de Rúbrica de Evaluación

Usa puntuación de 1-5 para cada dimensión:

- Fundamentación: ¿las respuestas están claramente vinculadas a las fuentes importadas?
- Calidad de citación: ¿las referencias a fuentes son visibles y específicas?
- Funcionalidad: ¿puede el usuario completar el flujo de pregunta-respuesta?
- Coherencia del producto: ¿el flujo de trabajo se siente integrado?
