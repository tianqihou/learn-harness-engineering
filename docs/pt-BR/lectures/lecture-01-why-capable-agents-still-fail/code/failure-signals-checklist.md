# Checklist de Sinais de Falha

Utilize este checklist ao revisar uma execução com um harness fraco.

- O agente perguntou, ou inferiu incorretamente, como iniciar a aplicação?
- Ele criou diretórios ou abstrações que não correspondem ao produto pretendido?
- Ele parou após criar apenas uma interface visual sem um fluxo completo funcionando?
- Ele deixou notas ou artefatos que ajudam uma futura execução a continuar o trabalho?
- Uma nova sessão conseguiria entender o que aconteceu em menos de cinco minutos?
