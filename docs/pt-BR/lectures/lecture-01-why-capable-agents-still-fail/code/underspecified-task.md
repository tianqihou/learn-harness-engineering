# Exemplo de Tarefa Subespecificada

Crie um aplicativo desktop de base de conhecimento com perguntas e respostas utilizando IA.

Restrições:

- Nenhuma especificada
- Nenhum comando de inicialização fornecido
- Nenhuma orientação sobre estrutura de pastas
- Nenhum modelo de dados definido
- Nenhum critério explícito de conclusão

Resultados típicos desse tipo de prompt:

- o agente inventa uma estrutura de forma improvisada
- o aplicativo pode compilar, mas não iniciar de forma consistente
- a interface pode aparecer antes de existir um fluxo utilizável de ingestão/consulta
- o agente frequentemente para após um sucesso apenas cosmético
