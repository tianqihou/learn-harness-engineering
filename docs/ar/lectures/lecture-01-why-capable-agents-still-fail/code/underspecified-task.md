# مثال على مهمة غير محددة بشكل كافٍ

بناء تطبيق سطح مكتب لقاعدة المعرفة مع إجابات أسئلة بالذكاء الاصطناعي.

القيود:

- لم يتم تحديد أي قيود
- لم يتم إعطاء أمر تشغيل
- لم يتم توفير إرشادات هيكل المجلدات
- لم يتم تعريف نموذج بيانات
- لا توجد معايير إكمال صريحة

النتائج النموذجية لهذا النوع من prompt:

- الوكيل يخترع هيكلاً بشكل مخصص
- التطبيق قد يُترجم لكنه لا يبدأ بشكل متسق
- الواجهة قد تظهر قبل أن يكون هناك مسار استيراد/استعلام قابل للاستخدام
- الوكيل يتوقف غالباً بعد نجاح تجميلي
