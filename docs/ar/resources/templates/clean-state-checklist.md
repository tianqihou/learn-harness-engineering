# قائمة فحص الحالة النظيفة

- [ ] مسار بدء التشغيل القياسي لا يزال يعمل.
- [ ] مسار التحقق القياسي لا يزال يعمل.
- [ ] التقدم الحالي مسجل في سجل التقدم.
- [ ] حالة الميزات تعكس ما هو ناجح فعلًا مقابل غير مُتحقَّق.
- [ ] لا توجد خطوة نصف منتهية بدون توثيق.
- [ ] الجلسة التالية يمكنها الاستمرار بدون إصلاح يدوي.
