# معيار المُقيِّم

استخدم هذا المعيار بعد التنفيذ وقبل القبول النهائي.

| الفئة | السؤال | النتيجة (0-2) | ملاحظات |
| --- | --- | --- | --- |
| الصحة | هل السلوك المُنفَّذ يتطابق مع الميزة المطلوبة؟ |  |  |
| التحقق | هل تم تشغيل الفحوصات المطلوبة فعلًا، مع أدلة؟ |  |  |
| انضباط النطاق | هل بقيت الجلسة داخل نطاق الميزة المختارة؟ |  |  |
| الموثوقية | هل تتحمل النتيجة إعادة التشغيل أو إعادة التشغيل بدون إصلاح؟ |  |  |
| قابلية الصيانة | هل الكود والتوثيق واضحان بما يكفي للجلسة التالية؟ |  |  |
| جاهزية التسليم | هل يمكن لجلسة جديدة الاستمرار من منتجات المستودع فقط؟ |  |  |
