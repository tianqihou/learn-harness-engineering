# وثيقة الجودة

لقطة جودة لكل مجال منتج وطبقة معمارية. يمكن للوكلاء والبشر على حد سواء استخدام هذه الوثيقة لفهم أين تكون قاعدة الكود قوية وأين تحتاج إلى عمل بسرعة.

**وتيرة التحديث:** بعد كل جلسة مهمة، أو قبل بدء مرحلة عمل جديدة.

**مقياس الدرجات:**

- **A**: جميع عمليات التحقق ناجحة، بنية نظيفة، قابلية قراءة الوكيل، اختبارات مستقرة
- **B**: التحقق ناجح، نظيف في الغالب، فجوات طفيفة في قابلية القراءة أو تغطية الاختبار
- **C**: يعمل جزئيًا، فجوات معروفة، بعض مناطق الكود صعبة على الوكلاء لفهمها
- **D**: لا يعمل، أو مشاكل هيكلية كبيرة
