# فهرس مواصفات المنتج

استخدم هذا المجلد لمواصفات سلوك واجهة المستخدم الحالية.

## المواصفات النشطة

- `new-user-onboarding.md`

## القواعد

- يجب أن تصف المواصفات سلوك المستخدم المرئي ومعايير القبول.
- إذا انحرف التنفيذ عن المواصفة، حدِّث أحدهما في نفس الجلسة.
- أبقِ هذا الفهرس محدثًا حتى يتمكن وكيل جديد من اكتشاف نطاق المنتج بسرعة.
