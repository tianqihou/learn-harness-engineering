# المعتقدات الأساسية

- المستودع هو سجل النظام المرجعي للوكلاء.
- `AGENTS.md` هو موجّه، وليس موسوعة.
- أدلة التحقق أهم من الثقة.
- مهمة واحدة محددة أفضل من العديد من المهام نصف المنتهية.
- التغذية الراجعة البشرية المتكررة يجب أن تصبح قواعد هارنيس قابلة لإعادة الاستخدام.
- التنظيف والتبسيط جزء من الشحن، وليس أفكارًا لاحقة.
- إذا لم يستطع وكيل اكتشاف حقيقة في المستودع، تعامل مع تلك الحقيقة على أنها غير متاحة تشغيليًا.
