# Exemple de Rubrique d'Évaluation

Utilisez une notation de 1 à 5 pour chaque dimension :

- Fondement : les réponses sont-elles clairement liées aux sources importées ?
- Qualité de citation : les références aux sources sont-elles visibles et spécifiques ?
- Fonctionnalité : l'utilisateur peut-il compléter le flux question-réponse ?
- Cohérence produit : le flux de travail semble-t-il intégré ?
