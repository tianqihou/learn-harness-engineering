# Liste de Contrôle du Système d'Enregistrement

Un nouvel agent peut-il découvrir ce qui suit uniquement depuis le dépôt ?

- Quel produit est en cours de construction ?
- Que doit faire l'application pour les utilisateurs ?
- Comment la base de code est-elle organisée ?
- Comment l'application démarre-t-elle ?
- Comment la santé est-elle vérifiée ?
- Quel travail est actuellement en cours ?
- Quelles normes de qualité sont importantes ?
