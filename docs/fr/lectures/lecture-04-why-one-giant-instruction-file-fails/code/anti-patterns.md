# Anti-Patrons des Fichiers d'Instructions

- Mettre toutes les connaissances du dépôt dans un seul fichier
- Répéter la même règle à plusieurs endroits
- Encoder des règles obsolètes que personne n'audit
- Écrire des instructions conditionnelles si spécifiques qu'elles s'appliquent rarement
- Intégrer de longs manuels d'outils dans le contexte de démarrage
