# AGENTS.md

## Commencez ici

- Lisez `docs/ARCHITECTURE.md`
- Lisez `docs/PRODUCT.md`
- Utilisez `npm run dev` pour démarrer l'application
- Utilisez `npm run check` avant de marquer le travail comme terminé

## Règles strictes

- Ne modifiez pas les limites main/preload/renderer d'Electron sans avoir lu
  `docs/ARCHITECTURE.md`
- Ne marquez pas une fonctionnalité comme terminée sans vérification
- Laissez un état propre (clean state) pour la prochaine session
