# Liste de contrôle d'état propre

- [ ] Le chemin de démarrage standard fonctionne toujours.
- [ ] Le chemin de vérification standard s'exécute toujours.
- [ ] La progression actuelle est enregistrée dans le journal de progression.
- [ ] L'état des fonctionnalités reflète ce qui réussit réellement par rapport à ce qui n'est pas vérifié.
- [ ] Aucune étape à moitié terminée n'est laissée non documentée.
- [ ] La prochaine session peut continuer sans réparation manuelle.
