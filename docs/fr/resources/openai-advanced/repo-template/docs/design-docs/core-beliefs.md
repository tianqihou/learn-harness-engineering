# Croyances fondamentales

- Le dépôt est le système de référence pour les agents.
- `AGENTS.md` est un routeur, pas une encyclopédie.
- La preuve de vérification importe plus que la confiance.
- Une tâche délimitée vaut mieux que plusieurs tâches à moitié terminées.
- Les retours humains répétés doivent devenir des règles de harness réutilisables.
- Le nettoyage et la simplification font partie de la livraison, ce ne sont pas des après-pensées.
- Si un agent ne peut pas découvrir un fait dans le dépôt, considérez ce fait comme opérationnellement indisponible.
